# 安装
1. 将数据包放入".minecraft/saves/(世界名称)/datapacks"
2. 输入命令"/reload"
# 卸载
1. 关闭游戏（服务端）并移除数据包。
2. 重启游戏(服务端)并输入命令"/reload"
# 警告
自游戏版本1.19起，轻量级分支版本"Disable_Vanilla_Lite"发布。该版本可能会与使用了"minecraft"命名空间来添加进度或配方的数据包之间产生兼容性问题。如果您遇到此类问题，请使用常规版本。
